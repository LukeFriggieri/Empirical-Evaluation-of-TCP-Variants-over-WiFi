# Empirical-Evaluation-of-TCP-Variants-over-WiFi
Reproducibility repository for "Empirical Evaluation of TCP Congestion Control Variants and a Kernel Module Amendment to TCP Veno over a Controlled AWGN-Impaired Wi-Fi Link" Contains raw data, statistical analysis scripts, kernel module source, and figures for the baseline TCP variant comparison and the Adaptive Beta/RTT Growth Guard validation
